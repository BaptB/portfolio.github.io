Optiplant

Optiplant est une interface web connectée à une base de données permettant de visualiser et gérer les paramètres d’une ombrière intelligente.

×

Fonctionnement

L’interface récupère les données via une API et affiche en temps réel l'humidité, la température, et la luminosité. Les utilisateurs peuvent consulter l’historique et ajuster les paramètres.

×

Technologies utilisées

HTML, CSS, JavaScript, PHP pour le backend, et une base de données MySQL. Les données sont visualisées à l’aide de bibliothèques comme Chart.js.
